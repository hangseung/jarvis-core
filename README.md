# Jarvis Core: AI Backend Assistant

Jarvis Core는 AI 기반 백엔드 어시스턴트 프로젝트입니다.

## 개요

이 프로젝트는 Google Gemini AI 모델을 활용하여 Slack 봇으로 동작하는 지능형 어시스턴트를 제공합니다.

## 주요 기능

- Slack 봇 통합
- Google Gemini 2.5 Flash 모델 활용
- 실시간 AI 응답 생성
- 멘션 및 메시지 이벤트 처리

## 설정

환경 변수가 필요합니다:
- `GOOGLE_API_KEY`: Google AI API 키
- `SLACK_BOT_TOKEN`: Slack 봇 토큰

## 사용법

프로젝트를 실행하여 Slack에서 AI 어시스턴트와 상호작용할 수 있습니다.
